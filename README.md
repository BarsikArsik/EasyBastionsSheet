
# EasyBastionsSheet
A module for Foundry VTT that adds an interactive DND2024 bastion sheet allows you to save data in a log and return to it for further editing. This is part of the EasyBastion module's functionality.
***
Модуль для Foundry VTT, добавляющий интерактивный лист бастиона DND2024, позволяет сохранить данные в журнале и вернуться к ним, чтобы продолжить редактировать. Это часть функционала модуля EasyBastion.

# Как это работает

В модуле добавлен интерактивный лист. Запуск осуществляется одноименным макросом "Лист Бастиона" и позволяет вести учет построек их особенности, отмечать количество защитников Бастиона и  так далее.  Для игроков доступен отдельный макрос, позволяющий просматривать Лист, но не редактировать его.
